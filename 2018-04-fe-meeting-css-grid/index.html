Waarvoor gebruik je Grid Layout?

  • Wanneer je elementen zowel verticaal als horizontaal wilt kunnen positioneren
  • Wanneer items zijn opgemaakt in een grid (producten, nieuws)
  • Complexe layouts met over elkaar gepositioneerde elementen (posters, headers, (typo)grafische layouts)

Waar is CSS Grid Layout goed in?

  • Elementen over 2 assen positioneren
  • Elementen elkaar laten overlappen
  • Layout op container niveau
  • Layout op basis van media queries
  • Kolommen aanmaken / laten passen adhv content
  • Consistente ruimtes tussen kolommen en rijen

Waar kan CSS Grid niet (goed)?

  • Elementen "buiten de lijntjes" plaatsen
  • Masonry-achtige layouts
  • Animeren van cell posities*
