CSS Grid Layout

What is Grid Layout?

A new way to define a two-dimensional layout on a page or element.

What does Grid do?

  • Splitting up a container in "cells"
  • Via rows and columns
  • To place and align content within those cells

Divide and conquer

Different ways to define "planes" on a grid:

  • Explicit rows
  • Explicit columns
  • Implicit rows
  • Implicit columns
  • Grid areas

Explicit rows & columns


          /* In full */
          .grid {
            display: grid;
            grid-template-rows: 1fr 200px auto;
            grid-template-columns: 1fr auto 1fr;
          }

          /* Shorthand */
          .grid {
            grid-template: 1fr 200px auto / 1fr auto 1fr;
          }

Naming things


          /* In full */
          .grid {
            grid-gap: 20px;
            grid-template-columns: 6fr 6fr 1fr 6fr;
            grid-template-rows: 1fr 2fr 1fr;
            grid-template-areas:
              "header header header header" 
              "main   main   .      sidebar"
              "footer footer footer footer" 
          }

          /* Shorthand */
          .grid {
            grid-gap: 20px;
            grid-template:
              "header header header header"   1fr
              "main   main   .      sidebar"  2fr
              "footer footer footer footer"   1fr
              / 6fr   6fr    1fr    6fr;
          }

Intrinsic sizing keywords

Intrinsic sizing depends on the content, instead of a parent.

min-content: Make the column as narrow as possible, fitting the content.

max-content: Make the column as wide as possible, fitting the content.

fr unit: Divides a grid into fractions.

Grid functions

minmax(200px, 1fr): Give the column/row a min and max size.

repeat(3, 1fr): Repeat the given size for an amount of columns/rows

repeat(auto-fit, minmax(100px, 1fr)): Fit the width/height of the container using the max size

repeat(auto-fill, minmax(100px, 1fr)): Fill the width/height of the container using the min size

When do you use Grid Layout?

  • When you need to lay out elements in two dimensions
  • Whenever you see a grid (products, tiles, etc.)
  • Complex layouts whenever you need elements on top of each other aligned in a grid (typographic posters)

CSS Grid is great at:

  • Position elements on 2 axes
  • Overlap elements withing a grid
  • Layout on container level
  • Layout based on media queries
  • Auto-create and -fit colums and rows based on content
  • Consistent spacing between cells

CSS Grid isn't good at:

  • Position element "outside the box"
  • Masonry layouts
  • Animating van cell positions*

* Not with CSS, but it can be done with JS üî•
